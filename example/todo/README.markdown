todo example
============

an example of the use of *tinyprofiler*

  * getting started
  * more information

getting started
---------------

install the project dependencies:

```bash
npm install --dev
```

compile the client bundle:

```bash
npm run-script build
```

start the server:

```bash
coffee server.coffee
```

visit the site:

```bash
firefox localhost:1337
```

----------------
